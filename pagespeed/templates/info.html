About Core Web Vitals

Web Vitals is an initiative by Google to provide unified guidance for quality signals that are essential to delivering a great user experience on the web.

Crawl Website Visit Offical Site

First Content Paint (FCP)

FCP is a user-centric metric for measuring perceived load speed. It marks the first point in the page load timeline where the user can see anything on the screen.

Find out more

Largest Content Paint (LCP)

LCP reports the render time of the largest content element (e.g., largest image or text block) that is visibible within the viewport.

Find out more

First Input Delay (FID)

FID measure the time from when a user first intereacts with a page (e.g., click on a link), to the time when the browser is actually able to respond to that interaction.

Find out more

Cumulative Layout Shift (CLS)

CLS measures the sum total of all individual layout shift scores for evey unexpected layout shift that occurs during the entire lifespan of the page.
