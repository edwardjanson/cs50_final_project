About

Core Web Vitals is an initiative by Google to provide unified guidance for quality signals that are essential to delivering a great user experience on the web.

The Core Web Vitals checker fetches website pages and provides an overview of FCP, LCP, FID, and CLS performance at page level. Please note that less popular URLs may not have sufficient data, resulting in the performance overview showing "No data".

First Content Paint (FCP)

Although not a core web vital, FCP directly affects the core web vitals like LCP. It marks the first point in the page load timeline where the user can see anything on the screen.

Largest Content Paint (LCP)

LCP reports the render time of the largest content element (e.g., largest image or text block) that is visibible within the viewport.

First Input Delay (FID)

FID measure the time from when a user first intereacts with a page (e.g., click on a link), to the time when the browser is actually able to respond to that interaction.

Cumulative Layout Shift (CLS)

CLS measures the sum total of all individual layout shift scores for evey unexpected layout shift that occurs during the entire lifespan of the page.
